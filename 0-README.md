# Food


## Ingredients
- put within `em` tags
- can put `buy` and `prep` classes on `li` or `span`

### Don't include "of"
- `2 sticks <em>celery</em>`
- `clove <em>garlic</em>`

### Quantity before em
- cloves before em  
  `2 cloves <em>garlic</em>`
- tin before em  
  `1 tin <em>chick peas</em>`

### Prep details after em  
- `<em>lettuce</em> finely chopped`
